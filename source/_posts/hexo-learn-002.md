### 创建

```bash
$ hexo init blog
$ cd blog
$ npm install
```

- \_config.yml
  配置信息

- package.json
  应用信息和依赖。

- scaffolds
  模板。Hexo根据scaffold来生成文件。

- source
  资源文件夹，用来存放用户资源。

- theme
  主题。Hexo根据主题来生成静态页面。
