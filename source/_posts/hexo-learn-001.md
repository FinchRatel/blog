### 什么是 Hexo？
Hexo 是一个快速、简洁且高效的博客框架。

### 安装
- 安装git
[Git for Windows 国内下载站](https://github.com/waylau/git-for-win)
- 安装nodejs
[Nodejs中文官网](http://nodejs.cn/)
- 安装hexo-cli
```bash
$ npm install -g hexo-cli
```
